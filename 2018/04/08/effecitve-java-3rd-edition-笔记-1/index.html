第二章 对象的创建与销毁

技巧 1: 使用静态工厂方法代替构造函数

静态工厂放法具有诸多优点：

  1. 见名知意，看到函数名就知道其功能，与所需参数。
  2. 构造函数每次调用都构造新的对象，但静态工厂方法不是。
  3. 可返回声明对象的子类对象实例。
  4. 根据传入参数不同，可以返回不同的对象实例
  5. 定义该静态工厂方法时，返回对象的类可以为尚未定义状态。

缺点：

  1. 仅仅使用这种方法的类不能被继承；
  2. 不便于代码使用者查找到该类方法，因为java文档对于这种技巧方法尚无特别支持。
